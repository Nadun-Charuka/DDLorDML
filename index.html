Planing a Database System


A database is a set of information that is in a structured form. This can be managed using traditional methods as well as computer technology. However, in the traditional method time, labor, and physical space are spent in the majority. With computerized management, the above-mentioned factors can be reduced, allowing for bookings with minimal error.
